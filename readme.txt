**Read Sample** add a customizable "Read Sample" button to WooCommerce product pages, allowing customers to view sample images in a popup. 

== Description ==

**Read Sample** add a customizable "Read Sample" button to WooCommerce product single page. This button allows customers to read any document sample before buy in a responsive popup area, enhancing their shopping experience by providing previews of the product sample before purchase.

Features:
- Upload multiple sample images files for each product using the WooCommerce product edit screen.
- Display a "Read Sample" button on single product page.
- Load sample content in a popup window via AJAX for a seamless user experience.
- Customizable settings for button text color, background color & button text.

== Installation ==

= Plugin Installation Method: =

1. Go to the WordPress dashboard
2. Got to the **Plugins** > **Add New**
3. Type **"Read Sample"** in the search box
4. Click on **"Install"** button
5. After install the plugin **Activate** the plugin

= Installation via Zip file: =

1. Download Read Sample **plugin zip** file from WordPress
2. Go to site **Dashboard** > **Plugins** > **Add New** > **Upload Plugins**
3. Select the `read-sample.zip`  file and proceed to install
4. Activate Read Sample.

= Plugin Configuration: =

1. Go to site **Dashboard** > **Tools** > **Read Sample Settings**
2. Go to the WooCommerce product edit screen to add sample images for your products.
3. Configure any additional settings as needed.

== Frequently Asked Questions ==

= How do I upload sample images? =

- After activating the plugin, go to the WooCommerce product edit screen. You will find a new field labeled "Sample Image Gallery" where you can upload multiple images using the media uploader.

= How can I customize the "Read Sample" button? =

- The appearance of the "Read Sample" button can be customized through the plugin settings. You can change the button text, color, and other styles to match your theme.

= Does this plugin work with all WordPress themes? =

- While the plugin is designed to work with most WordPress themes, some custom themes may require additional styling adjustments.
